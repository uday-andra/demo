Selector:

It is used to select HTML elements for styling.
  • element Selector
  • id selector
  • class selector
  • attribute selector
  • grouping selector
  • Universal selector
  • combined Selector
    • descendant selector
    • child selector
    • adjacent sibling selector
    • general sibling selector

Element Selector:

It is used to select elements by their tag name.
        syntax:
                tagName{
                    property:value;
                }

Id selector:

It is used to select elements by their id.

id is an attribute used in opening tag.

id should be unique.

id selector indicated with "#".

        syntax:
                #idName{
                    property:value;
                }

Class selector:

It is used to select elements by their class.

Class is an attribute used in opening tag.

Class name should be duplicate.

Class selecto indicated with "." . 

        syntax: 
                .className{
                    property:value;
                }

Grouping selector:

It is used to select elements as grouping.

Class selecto indicated with "," .

It is used to apply same style to multiple elements.

        syntax: 
                selector1,selector2,selector3{
                    property:value;
                }

Universal Selector

It is used to select all elements

Universal selector indicated with *

        syntax:       *{
                        property: value;
                    }
        Note : it is recommended to have 
        padding :0;
        margin:0;
        box-sizing: border-box;
        content-box-it is used to set the box model by default
                   -it calculates the width and height of an element
                    including padding and border
                    ex: width: 100px;
                        height: 100px;
                        padding: 10px;
                        border: 2px solid black;
                        box-sizing: content-box;
                    calculation:
                        width: 100px + 10px + 2px = 112px;
                        height: 100px + 10px + 2px = 112px;

     box-sizing: border-box-it is used to set the box model by manually
                -it calculates the width and height of an element 
                including padding and border and provides the exact width and height
                ex: width: 100px;
                    height: 100px;
                    padding: 10px;
                    border: 2px solid black;
                    box-sizing: border-box;
                calculation:
                    width: 100px + 10px + 10px + 2px + 2px = 124px;
                    height: 100px + 10px + 10px + 2px + 2px = 124px;

Combined Selector:

It is used to combine multiple selectors

It is used to apply same style to multiple elements

Descendent Selector:

It is used to select elements that are descendent of a specified element

Used to select both direct and indirect child elements

descendant selectror indicated with a space

            syntax:
                    parent child{
                        property:value;
                    }

Child selector:

It is used to select elements that are direct children of a specified element

used to select only direct chiled elements

child selector indicated with > symbol

            syntax: 
                    parent > child{
                        propert: value;
                    }

Adjascent Sibling selector:

It is used to select elements that are Adjascent Sibling of a specified element

used to select only next sibling elements

adjascent sibling selector indicated with + symbol

            syntax: 
                    element1 + element2{
                        propert: value;
                    }

General Sibling selector:

It is used to select elements that are General Sibling of a specified element

used to select next all sibling elements

general sibling selector indicated with ~ symbol

            syntax: 
                    element1 ~ element2{
                        propert: value;
                    }
